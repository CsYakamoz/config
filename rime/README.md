## Configuration Directory

Mac: `~/Library/Rime`

Linux: `~/.config/ibus/rime` or `~/.config/fcitx/rime`

Windows: `%APPDATA%\Rime`

### Mac

`System Preferences` -> `Keyboard` -> `Input Sources` -> `Automatically switch to a document's input source`
